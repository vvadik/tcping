Пример запуска:

sudo python3 tcping.py google.com 1

---------------------------------------------------------------------------------------
Аргументы запуска:

sudo python3 tcping.py [host] [i] [-n attempts] [-p port] [-w wait sec] [--from-ip 127.0.0.1] [-d True]

host IP или URL хоста
i Интервал между отправкой пакетов
-n количество запросов к хосту, по умолчанию бесконечный пинг
-p порт на который будет отправлен запрос, по умолчанию 80
-w Ожидание ответа в секундах, по умолчанию 2 
--from-ip Возможность указать интерфейс, отличный от найденного по умолчанию
-d Debug режим, при котором отображается байты пакета + их разбор
